Real Estate Price Analysis Dashboard

Project Overview
This project is an interactive Tableau dashboard that analyzes real estate data. It provides insights into:

Revenue trends over time
Average price per bedroom
Price distribution by Zip Code
Distinct count of bedrooms in listings

Problem It Solves
Real estate professionals and buyers often struggle to identify Pricing trends across different regions and property sizes,How bedroom count impacts property prices and Market performance over time.

This dashboard solves that by offering clear, visual insights into property pricing and trends, enabling better decision-making for pricing strategies and investments.


Tools Used

Tableau Public for data visualization and dashboard creation
Data Cleaning & Preparation (CSV/Excel as source)
Calculated Fields & Filters for dynamic analysis


Key Features

Line chart showing Revenue for the Year
Bar chart for Average Price per Bedroom
Color-coded bar chart for Price by Zip Code
Interactive filters for bedrooms and zip codes
